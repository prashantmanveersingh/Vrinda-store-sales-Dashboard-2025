# Vrinda-store-sales-Dashboard-2025
This project presents an interactive Excel dashboard created using Pivot Tables and Pivot Charts to analyze and visualize business data. The dashboard provides meaningful insights through dynamic reports, making it easier to monitor key performance indicators and identify trends.

**Objective:**
Vrinda store wants to create an annual sales report for 2025. So that, Vrinda can understand their customers and grow more sales in 2026

**Sample Questions:**
1.  Compare the sales and orders using single chart
2.  which month got the highest sales and orders?
3.  Who purchased more- Men or Women in 2025?
4.  What are the different order status in 2025
5.  List top 5 states contributing the sales ?
6.  Relation between age and gender based on number on orders?
7.  which channel is contributing the max. sales?
8.  Highest saling category?
